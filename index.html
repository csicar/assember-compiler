Assembler Befehle

momentan unterstützt der Compiler folgende Befehle:

  • MOV [Register],[Register]
  • CPY alias von MOV
  • MOVN [Register],[Zahl in Bin / Dec / Hex]
  • AX TO CX
  • VALUE TO [Register]
  • NO-OP
  • ALU ENABLE [ADD / SUB / INCDX / INCBX / DECDX / DECBX / AND / XOR / OR]

Die Befehle werden über Regular Expressions gecheckt.
